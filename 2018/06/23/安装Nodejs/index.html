安装Nodejs

Share

1.下载nodejs并安装

2.设置全局路径

这样下次用npm install -g 全局安装时会把包安装到此目录下

3.查看设置是否生效

4.安装taobao NPM镜像

因为第2步已把全局路径改成了”E:\nodejs”，所以cnpm会被安装到这个路径下

5.配置cnpm环境变量

将”E:\nodejs”配置到windows的环境变量中

6.安装成功
